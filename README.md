# Ansible Inventory OKD 3.11

## Ansible

### Run

```bash
# Prepare
ansible-playbook -i hosts.yaml \
  openshift-ansible/playbooks/prerequisites.yml

# Deploy
ansible-playbook -i hosts.yaml \
  openshift-ansible/playbooks/deploy_cluster.yml

# Uninstall
ansible-playbook -i hosts.yaml \
  openshift-ansible/playbooks/adhoc/uninstall.yml
```
